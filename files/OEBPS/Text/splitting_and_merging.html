Splitting and Merging

— Working with Chapters and Files —

An EPUB can consist of one large HTML file with all of your text, or multiple files each containing part of your book. In most EPUBs each main Chapter is stored in its own HTML file in the EPUB. This isn't required, but it makes it easier to work with the book and can make reading and editing quicker since the files are smaller. All of your files can be accessed using the Book Browser.

If you have loaded one HTML file into Sigil and want to divide it into separate files, then you can use Sigil's Split functions. And if you have several smaller files that you want to combine into a larger file, then you can use the Merge functions.

Tip

Most ereaders start each new HTML file on a new page – giving you automatic page breaks between chapters if you split your Chapters into separate files.

Splitting Files

You can split a file into multiple files in a couple of ways.

Split Immediately

The easiest way to separate a file into two files is with the Edit→Split Chapter menu item, or the Split Chapter icon:

To use Split Chapter, click at the end of paragraph where you want to do the split:

Then use Edit→Split Chapter:

Everything from the beginning of the original file up to and including the paragraph where you clicked is put into one file, and everything after the paragraph you clicked on was put into a second file. Note that the first file kept its name, and a new file with a new name was created to hold the second file. You can rename and edit the files as necessary.

Tip

The split is done at the end of the paragraph you clicked in.

Mark and Then Split

If you want to split a very large file into many chapters, then it is probably easier to use the chapter marking approach.

Scroll through your file, and wherever you want a new chapter to start click the beginning of the chapter name or the end of the previous paragraph. Then use Edit→Insert Chapter Marker. This will insert code into your document telling Sigil you want a new chapter to begin here – you will see a horizontal line in Book View exactly where you inserted the marker:

When you have marked all your chapters, use Edit→Split at Chapter Markers. This will automatically split your file into as many chapters as you marked. If you decide not to split the file, you can delete the horizontal marker lines.

Once you have split your files you rename them. And you can even Merge some or all of them together again.

Tip

If you split a file, any header information is copied to the new file.

Merging Files

If you need to combine two or more files into one file, you can use Merge.

In the Book Browser select the files you want to merge, then right click and select Merge from the context menu (or use the keyboard shortcut).

Your files will be combined into the first file.

Tip

If you only have one file selected and use Merge, then the selected file will be merged with the previous file in the Book Browser.
