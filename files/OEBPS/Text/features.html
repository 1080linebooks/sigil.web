Features

— An In-Depth Guide —

This section gives a quick overview of all of Sigil's features followed by a detailed description of everything available in Sigil.

You can also read the Tutorials chapter for step-by-step instructions on common tasks in Sigil, and then refer back to this section for additional details.

Editing

You can edit your book using either Book View or Code View. The Book View editor is a WYSIWYG editor which displays your book as it might appear to users and is useful when you need to make basic changes. The Code View editor allows you to directly edit the HTML formatting of your files when you want complete control over your book or you want to use Sigil's advanced features. There is also a read-only Preview View which allows you to test modifying styles and to inspect your code in detail. Tabs are opened for each of the files in the EPUB that you edit, and you can even magnify the text. Spell checking with multiple dictionaries is also available.

Find & Replace

The ability to search for or replace text is one of the most powerful functions in Sigil. Book View provides a basic find function, but Find & Replace is usually used in Code View where all its features are available and it can be used to change or correct formatting and styles. You can search or replace in one file or in all the files at once, and you can search for simple text or use Regular Expressions.

Managing Files

The Book Browser organizes all your files by type and gives you quick access to open files or to modify them. Double-click a file to open it in the editing window, or right-click on selected files to see options for renaming, deleting, merging, adding, linking stylesheets, etc. The HTML files are listed in the order they appear in your book – and you can drag and drop them to reorder them.

Table of Contents

The Table of Contents (TOC) window shows you exactly what your TOC will look like, and allows you to jump to specific chapters by clicking on chapter names. You can also generate a new TOC based on headings in your book at any time, and create an HTML version of your TOC.

Modifying Book Details

The Metadata Editor provides a quick way to update book details like Title, Author, and Language. You can add any of the hundreds of standard EPUB metadata entries like publication date, co-authors, copyright information, etc.

Preferences

Many settings in Sigil can be customised by opening the Preferences window. In particular you can set Sigil's user interface language, keyboard shortcuts, text appearance and dictionaries.

Validation

Sigil performs automatic checking of your code to highlight errors when saving, and it includes the FlightCrew EPUB validator to help ensure your book conforms to EPUB standards. Sigil can also automatically clean and format your code using HTML Tidy.

Advanced Tools

Sigil also provides additional tools to assist with editing - including an Index Editor, a Search Editor to save your Find & Replace searches, a Clip Editor and Clipboard History to save text.  Several reports are also available to display details about your files and styles.

Language Support

Sigil has been translated into over 15 languages and makes it easy to change the user interface language as needed. It is written in C++ for high performance and includes full Unicode UTF-16 support.
