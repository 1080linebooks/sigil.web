Index Editor

— Create an Index For Your Book —

If you need to create an Index for your book, you can use the Index Editor to help.

The Index editor is used to manage a list of entries that should be in the index, and then use that list to create an HTML page for your index.

You can add words to the index in two ways.

One is to add just specific words or phrases in the book to the index when the index is created.  You can do this in Code View by selecting the text and using Tools→Index→Mark for Index.  This adds a class attribute that tells Create Index to add this text.

The other option is to add the words or phrases to the Index Editor, and then every occurrence of that word or phrase will be added to the index.

Tooltip information in the Index Editor provides additional information.

Use the right click context menu to add/modify entries.

The AutoFill option is just for amusement and adds every unique word in the book to the index.
