What's New

Sigil 0.6.0 contains an enormous amount of bug fixes and new features.  Almost all areas of Sigil's code were updated and improved to focus on:

  • Stability: All known crashes from previous versions have been identified and fixed - making Sigil completely stable for updating your EPUBs.
  • Robustness: Sigil will no longer exit when reading most invalid EPUBs and will even attempt to correct common issues so that the files can be opened.  Splits and merges can now be done exactly where you want without potential data loss, and your cursor position is accurately updated when you switch views.
  • Performance: Sigil is now significantly faster in many areas: opening and changing tabs, generating a TOC, merging files, bulk renaming, replacing text, etc.

New features include:

  • Saved Searches: Never forget a frequently used Find & Replace search again. Edit and group searches, run a group of searches, and quickly access searches using right-click in Find & Replace text boxes.
  • Clips: Store and easily access common text or styles. Edit and group clips, apply starting/ending tags to text, and quickly access clips using right-click in Book View or Code View editors.
  • Indexes: Create an HTML Index by marking text or listing phrases to index.
  • HTML Table Of Contents: Convert your TOC into an HTML page.
  • Edit Files Externally: Open your HTML, Image or Stylesheet files with an external application.  Sigil will update your book as soon as you save in the other application.
  • Go to links or styles: Quickly jump to the CSS style used for your HTML, and go Back to your starting point.
  • Reports: See details about your files, images, styles, including word counts and image sizes.
  • Delete Unused Images and Styles: Automatically remove unlinked images or unreferenced CSS classes.
  • Clipboard History: Quick access to your recently copied text
  • Spellcheck: Step through misspelled words in addition to highlighting them, with improved word recognition.
  • Preferences: Change Sigil's fonts and level of automatic source cleanup.
  • Special Characters: Insert from a list of common special characters. 
  • Insert Hyperlinks and IDs: For footnotes or links to chapters or anything else.
  • Link stylesheets: Quickly link or unlink stylesheets to your HTML files.
  • Validate CSS stylesheets: Check your stylesheet pages using the W3C web service with a simple click.
  • Edit headings while using Generate TOC: Change the title and the level of any heading entry when using Generate TOC for quick corrections.  Use the tooltips to see the actual HTML code for your heading. 
  • Find & Replace updates: Use the new Replace button to replace without moving to the next match, or the new options to make using regex easier (click on the chevron icon on the lower left of Find & Replace). 
Note:
  • Preview View has replaced Split View because Split View was a major cause of issues with older versions, and to allow more in depth inspection of code and styles. 
  • Clean Source defaults to Pretty Print instead of HTML Tidy.

Other changes/features include:

  • Support in Code View for using the formatting buttons.
  • Change Case - buttons to change the case of words
  • Insert Image improvements - insert directly from disk, display thumbnails, image size details
  • Metadata Editor redesign for simpler access and more control
  • Automatic Well-Formed Checking - your code is now checked for errors only when necessary so you no longer need to turn it off then on to do complex replaces.
  • Updated icons and streamlined menus for easier access to functions
  • Save a copy of your EPUB, or save individual files from your EPUB
  • Easily insert closing tags or delete tags
  • Show tag - while in Book View use show tag to quickly see the HTML code
  • And numerous other tweaks and polishing to make Sigil easier and quicker to use.
  • Updating of all third party software, including Qt 4.8.3
  • Ctrl-click on an opening tag in Code View to jump to the link or style class.
  • Hold the Ctrl key while clicking Find/Replace/etc.  (in Find & Replace or Saved Searches) to temporarily use Current File
  • Hold the Ctrl key while clicking a Special Character to close the dialog after inserting the character.
  • Double-click a tab header or use F2 to toggle between Book View and Code View.
  • Right-click on a tab header to select close all other tabs.
For a list of the changes made to the code, see the software Changelog. All of the new features, along with all of the original features and functionality are of course described in this User Guide.
