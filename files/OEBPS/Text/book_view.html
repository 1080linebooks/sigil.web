Book View

— WYSIWYG Editing —

Book View (or WYSIWYG mode – what you see is what you get) is one of two ways you can edit your book in Sigil. Book View is what most people use when they start to use Sigil since its the closest to a standard word processor. Its a good way to make basic changes to your book when you aren't too concerned about the underlying HTML code. However, when you want more control over the format of your EPUB or to use Sigil's advanced features then you can use Code View. 

You can switch between Book View and Code View at any time - using the buttons on the toolbar, the View->Toggle View State menu entry, the keyboard shortcut for toggle, and by double clicking on the tab name of the current tab.

To switch to Book View use View→Book View or its toolbar icon or keyboard shortcut:


Book View will remember your cursor position in your file if you switch to another file or if you change to another view and back to Book View.

Note

Certain features are only available in Code View.  For instance Spellchecking and the Search Editor can only be used in Code View.  Find & Replace will automatically switch you to Code View if you search across HTML files.

You may need to toggle to Code View and back to Book View to ensure your text is display as you expect if you make a number of paragraph changes that include additional styles, etc.

Book View will do its best to apply the style you intended, but occasionally it will guess wrong and you will need to reformat it. It may even be necessary to use Code View to adjust the formatting.

Delete an image or a blank line by placing your cursor just after the image or line and hit backspace - this preserves the formatting of the surrounding text.

Sometimes parts of the text are highlighted even though they are not selected. This is a bug in the Qt libraries used by Sigil and cannot be avoided.  The easiest workaround is to select an area around the highlighted text.

Context Menu

The Book View context menu can include a number of useful features.

Right click on an Image to see options for

  • Open imagename:  open the image tab for the image (also shows the image's name)
  • Open With: open the image in an external program for editing
  • Save As: save the image to a selected location on your computer
Right click on text to see options for:
  • Insert Image: insert an image where you clicked from your book or computer
  • Cut
  • Copy
  • Paste
  • Select All
