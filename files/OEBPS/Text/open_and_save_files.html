Open and Save Files

— Load and Save Your Books —

Sigil supports opening EPUB files and importing HTML files. However, Sigil always saves books as EPUB files. The EPUB file will contain all your HTML files, images, stylesheets, fonts, etc. zipped up into one file.

Tip

You should save your file immediately after opening in Sigil for the first time so that Sigil can check for errors and convert your book to EPUB.

Importing and Adding Documents

Open Files

To open a file, use File→Open and select the file you want to open:

EPUB Files

  • Everything in the EPUB file is loaded automatically – HTML files, images, stylesheets, metadata, etc.
  • Sigil will attempt to organise your files into standard folders as shown in the Book Browser.
  • Sigil will also attempt to clean up the EPUB for spacing and minor issues depending on your preferences.

HTML Files

  • The file and and linked images and stylesheets are imported if possible and placed into the appropriate folders.
  • The metadata in the HTML file will be loaded into the metadata for the book.
  • Sigil will attempt to cleanup the HTML file for spacing and minor issues depending on your preferences.

Add Files

To add files to your EPUB, use File→Add→Existing Files or the toolbar button:

HTML Files

  • Metadata is not loaded from the file (unlike when you use Open).
  • Sigil will attempt to load images that are referenced.
  • Files will be placed into their appropriate folder.

Image Files

  • Files will be placed into the Images folder.

Stylesheet Files

  • Files will be placed into the Styles folder.

Font Files

  • Files will be placed into the Fonts folder.

Other Files

  • All other files will be placed in the Misc folder as they are not usually used in an EPUB. Only certain text files in the Misc folder can be edited in Sigil.

Note

If there are any errors while opening your file, Sigil may fail and may give you a warning. Sigil cannot read EPUB files or documents that are very badly formatted or non-standard. You may need to open the file (using standard zip programs like WinZip or 7Zip) and correct the errors before reloading it into Sigil.

Add Blank Files

You can also create new empty files in Sigil.  Use File->Add and choose from blank HTML page, blank CSS stylesheet, or blank SVG image (a special image format).

Saving Your Book

Use File→Save or Ctrl+S to save your book. You should always save your document before doing major edits in case you want to undo them.

Tip

If you edit any text in your document, an asterisk ( * ) is added to the file's name in the titlebar.

When you save your book Sigil will check it for errors and warn you if you need to fix anything.

Sigil will not save a corrupt EPUB, so frequent saves will make it easier to recover from any unexpected errors. Sigil will also clean your source code according to your preferences when saving the file. See the Validate and Clean chapter for more details.

Note

Your saved EPUB file contains all the files that make up your book - your text, images, stylesheets, embedded fonts, table of contents, metadata, etc.

Save a Copy

You can use File→Save A Copy  to save a copy of your EPUB to a different filename.  The difference to Save or Save As is that the current file you are working on is not renamed.  So you can use this feature to make a backup of the file without changing the name of your file or the directory you save files to.

Saving Files

You can also right click on a file name in the Book Browser and select Save As from the context menu.  You can then name and save the specific file to a directory on your computer.  If you select more than one file, all files will be saved to the selected directory.

Editing Files Externally

Sigil is a powerful editor, but it can't edit images, and it may not have some of the features you use in other editors.  In these cases you can edit the file in an external program and when you save, Sigil will update its copy.

To edit a file externally, right click on the file's name in the Book Browser and choose Open With from the context menu.  You can also right click on an image in Book View to edit it directly.

When you open a file you first need to choose the application to use with that file type.  You can give the application a friendly name that will be used in the menu the next time you use Open With.  When you next use open with on the file you will see the friendly name you gave the application, and the option to choose a different application.  Only one default application will be listed but you can change it at any time.

Tip

You can edit your CSS stylesheets externally with a text editor while your book is displayed in Sigil. When you save in your externally application, Sigil will automatically update the layout of your book.

Note

If you make changes to the file in Sigil while the file is being edited in another program then you may lose data.  You should always open the file for editing externally, make the changes, then save and close the file in the external application before making any changes to that file in Sigil.
