Code View

— Direct HTML Editing —

Code View allows you to directly edit the HTML code that makes up your EPUB.

To switch to Code View use View→Code View or its toolbar icon or keyboard shortcut:

In addition to displaying your HTML code exactly as it is stored, Code View highlights tags and elements in colors to make it easier to read and identify issues. Code View will remember your cursor position for your file if you switch to another file or if you change to another view and back to Code View.

Tip

When switching to Code View from Book View or Preview View you can quickly find the same location in the text by highlighting the text in Book View, using Ctrl-F to save the text into Find & Replace, switch to Code View and then use Ctrl-G to find the text.

Code View gives you access to several features described elsewhere in the User Guide such as Replace in Find & Replace and Spell Check (words that are misspelled are underlined in red if this feature is active.

Tip

When switching from Code View to another window or saving your file, Sigil will check if your code has any major errors if the Well-Formed Check is enabled. If there are any errors you will be prompted to correct them Manually or Automatically. You should usually choose Manually to fix the issue to avoid any chance of the Automatic correction deleting text.

Note

If you have any entities in your code, e.g. &#257; and you make any changes while in Book View, then when you switch back to Code View you will see that Book View converted most of those entities into their actual character, e.g. ā. So after editing in Book View your entities will be shown as characters even when viewed in Code View.
