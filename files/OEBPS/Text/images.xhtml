Images

— Inserting pictures into your book —

You can insert images and pictures into your book – but first you must add them to your EPUB and then tell Sigil where they should be used in the ebook.

Adding Images to Your EPUB

To add images to your book, use File→New→Add Existing Files, or right click in the Book Browser and select Add Existing Files from the context menu. This will display a dialog allowing you to select one or more files from your computer. Select as many images as you want.

The images will be added to the Images folder in the Book Browser. However, they are not yet actually used anywhere in your book – they are just available to use.

Tip

To view an image's details double click on it in the Book Browser. The Width and Height in pixels, size of the file in kilobytes, and whether or not the image has any color defined are shown:

Note

If you delete and re-add an image you will need to save and re-open Sigil to display the new image.

Inserting Images Into Your Book

Once the images are in your EPUB, you can insert them anywhere in your files.

Click on the location in your file where you want the image to be located (in either Book View or Code View) and then use Edit→Insert Image or the Insert Image icon in the main toolbar.

Select the image you want to insert.

Tip

You can insert the same image into your text as many times as you want.

In Book View you can double click on an image or right click and select Image Properties to see information on the image link.

Cover Images

Images for covers are special in that you must tell the EPUB which image should be used for the cover. Different ereaders use different methods to identify the cover, so you should use both methods:

  • Create a cover.html file at the start of your book and then insert your image into it.
  • Mark the cover.html file as a cover file by right clicking on it in the Book Browser Text folder, selecting Add Semantics and choosing Cover.
  • Mark your actual cover image file as a Cover by right clicking on it's file in the Book Browser Images folder, selecting Add Semantics and choosing Cover.

Note

Please make sure you have permission to use any images in your book as most images are copyright and their use may require a license.
