Search Editor

— Saved Searches for Find & Replace —

The Search Editor complements Find & Replace by providing a way for you to save and run all your common searches.

Right Click on the Find text box to see a list of saved searches.

The Search Editor dialog can be used to manage your search entries (use right click context menu for most options).  It can also be used to run Find & Replace commands if you want a separate window for Find & Replace.
