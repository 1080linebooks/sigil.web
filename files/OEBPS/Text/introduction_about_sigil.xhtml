About Sigil

Sigil is used to create EPUB ebooks from HTML files and to modify the formatting of existing EPUBs. It is used by both individuals cleaning up a few ebooks for personal use and by professional editing companies publishing books on multiple platforms. Although you can use Sigil with its WYSIWYG editor to write your book, most authors using Sigil typically use their favorite editor for writing and then use Sigil to edit it to achieve the appearance they want and to package it into an EPUB.

Sigil was created by Strahinja Markovic in 2009 as completely free and open-source software. John Schember is currently the lead developer and has been since 2011 when he volunteered to become the new maintainer of Sigil. Other volunteers contribute code, translations and documentation to Sigil. See the Help→About menu in Sigil for a complete list of contributors.

This User Guide was created entirely in Sigil – you can load the EPUB version of this guide into Sigil to see an example EPUB ebook.
