Spell Check

— Checking and Correcting Spelling —

Sigil provides several ways to spell check your book. You can choose the main dictionary to use, and create personal dictionaries with your own words.

Note

Spell Check is only available in Code View.

Configuring Spell Checking

You can set the spell checking preferences that you want – including whether or not to use automatic spell checking, which main dictionary to use, and which custom dictionary to use if any. See Spell Checking under Edit→Preferences to change the available options.

Tip

Since you can use multiple user dictionaries, you can setup user dictionaries for different types of books or individual books. Then you can change user dictionaries as necessary.

Automatic Spell Checking

Code View will highlight any misspelled words with a wavy red underline when automatic spell checking is enabled in Preferences:

You can then right-click on the misspelled word to bring up a list of choices:

  • Suggested Words: a list of alternative words to choose from – selecting one will replace the misspelled word
  • Add to Dictionary: add this word to your current user dictionary
  • Ignore: ignores the spelling of this word in the entire document until you restart Sigil

Finding Misspelled Words

You can also search for misspelled words one at a time in order to make finding them easier.

In Code View, open the Find & Replace window and change the mode to Spell Check and clear any text in the Find box:

To search for misspelled words:

  • Click on Count All to display the number of misspelled words in your selected files.
  • Click on Find to highlight the next misspelled word. You can then right-click on it as above to change it or add it to the dictionary.

You can also enter a Regex search pattern in the Find box to find misspelled words that match the pattern you entered in the Find box.
