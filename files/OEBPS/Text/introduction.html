Introduction

BETA

— Welcome To Sigil: The EPUB Editor —

Sigil gives professional and amateur publishers, editors and authors the ability to precisely control the design and appearance of EPUB ebooks. It is completely free to download to use.

The Sigil User Guide includes documentation that will help you get the best out of editing your EPUB, including:

  • Installation: installing Sigil on Windows, MAC and Linux
  • Features: the complete description of all of Sigil's features
  • Tutorials: step by step guides for common activities in Sigil
  • FAQ: frequently asked questions about Sigil
  • Contributing to Sigil: how to support Sigil

Software and documentation is available online:

  • Sigil Official Website
  • Sigil Software
  • EPUB and mobi versions of the Sigil User Guide
  • HTML version of the Sigil User Guide

Help with Sigil is available from other users at MobileRead:

  • Sigil Forum @ MobileRead
