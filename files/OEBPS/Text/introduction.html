Introduction

— Welcome To Sigil: The EPUB Editor —

Sigil is the best completely free EPUB editor available. It is designed to provide professional and amateur publishers, editors and authors the ability to precisely control the design and appearance of EPUB ebooks.

Sigil software and documentation is available online:

  • Sigil Official Website
  • EPUB and mobi versions of the Sigil User Guide
  • HTML version of the Sigil User Guide
  • Sigil Software

Help with Sigil is available from Sigil users at MobileRead:

  • Sigil Forum @ MobileRead
