Add a Cover Image

Every book should have a cover image, and Sigil can help you add one to your EPUB. You will need to have an existing cover image available to use (and have permission to use it). The exact size of the cover is up to you and the guidelines for your device.

There are many ways to modify how the cover is displayed, so these instructions will just describe a basic cover. See covers in other EPUBs for examples.

To setup an EPUB cover you will need to follow these steps:

  • Create a new HTML page to hold your cover.
  • Insert your cover image into the HTML page.
  • Mark your image file and HTML file as covers.

As part of this tutorial you will also learn how to insert images, add files to your EPUB, rename files, and access more commands using context menus.

Create an HTML File For Your Cover

In order for the EPUB to know where in your book to put your cover, you need to create a new HTML file in your EPUB then insert your cover image into the file.

You can create a new HTML file in your EPUB by right-clicking on a file name in the Book Browser and selecting the menu item Add Blank HTML File:

A new HTML file called Section0001.html will be created in your Text folder:

Rename the file so you know what it is by right-clicking on the filename and selecting Rename from the menu:

Enter Cover.html as the new name (it must be "cover" or "Cover") (you can just type Cover – the extension will not change by default):

Chapters and pages are shown to the reader in the order they appear in the Book Browser.

Of course, since a cover should be at the front of the book, you need to move the cover file to the top of the Text folder.

You can drag-and-drop the file to its new position (click on the filename and hold the mouse down while you move it to its new position):

Insert Your Cover Image

Now that you have an empty HTML file, you need to put your cover image into it.

Open the HTML page you created to hold the cover by double-clicking on the Cover.html file. This will open it in the Book View editor window and you will see Cover.html as the title of the open tab in the editor window.

Click somewhere in the blank window to start editing:

Click on the Insert Image button on the toolbar. This will display the Insert Images window:

The Insert Images window shows you a list of images already in your EPUB. But since you haven’t loaded any images yet, it’s empty.

You can insert the image into your EPUB and into your HTML file in one step by clicking on the Other Files button.

Select your cover image from your filesystem and click OK.

Your cover image will now appear in the file:

Notice that your image file is now shown in the Book Browser:

You can double-click your image file to display it in the main window.

There are still a few steps needed to identify your image as a cover in the EPUB.

Mark Your Cover Files

Now that you have both a cover image and an HTML file with a cover in it, you need to mark both as Covers to make sure the various ereaders can recognise the cover (some readers just look for the cover image file, and some look for the HTML file containing your image).

To mark the HTML file as a cover right-click on cover.xhtml in the Book Browser window, select the menu item Add Semantics, then choose the Cover entry:

And to mark the image as a cover image right-click on cover.png file in the Book Browser window, select the menu item Add Semantics, then choose the Cover entry.

You must set both the image file and the HTML file to type Cover to make sure all ereaders recognise the cover.

Your EPUB now has a cover defined (if you look at the Add Semantics menu again for the files you will see the Cover items now have a checkmark next to them).

You should Save your book.
