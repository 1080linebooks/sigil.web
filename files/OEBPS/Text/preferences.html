Preferences

— Customize Settings —

The Preferences dialog gives you control over a number of settings in Sigil like Languages and Dictionaries.

Select the menu item Edit→Preferences to open the Preferences dialog.


Choose the area you want to modify by clicking on the entry on the left. See below for details on what can be changed.

Select the Close button to save the current settings and close the window.

Appearance

You can modify the fonts and colors that Sigil uses to display your book in Book View and Code View.

Note

These settings have no effect on your EPUB or on the appearance of your book on ereaders.

Keyboard Shortcuts

You can view all the keyboard shortcuts assigned to functions in Sigil – and change them to what you prefer. So if you want F2 to always open Find & Replace then you can change it here.


To set or change a shortcut:

  • Select the entry from the list
  • Click in the Shortcut text box
  • Type the key you want to use - do not spell it out using letters, just press the key sequence
  • Click on Assign to set the shortcut (if there is a conflict with another entry you will see a message)
  • You can also click on Remove to remove the shortcut - this will remove any shortcut for this entry

Use the Filter text box to easily find keys or descriptions that contain letters you type.

Use Reset All to change all shortcuts back to their defaults.

Language

You can set the language of Sigil's interface and the default language of your book:


User Interface Language

Sigil's user interface has been translated into many languages. You can select from the available languages using this option. If a translation is not available for specific words then they will be shown in English.

Note

You must restart Sigil in order for the new user interface language to be used.

Try setting the interface to another language and restarting Sigil to see the translations volunteers have created. You can easily change back to your preferred language.

Default Language For Metadata

This option sets the default language that will be selected in the Metadata Editor for new books or books that do not already have a language set.

Spell Check

Sigil supports spell checking in Code View, and allows you to choose from standard or personal dictionaries:


Enable Automatic Spell Checking

Standard Dictionaries

Sigil includes several dictionaries as standard: US English, British English, French, German, Spanish. You can select one using the Dictionary drop-down menu.

If you want to see where the standard dictionaries are stored, or to add new standard dictionaries (which must be in the hunspell format) then you can click on the "Open Dictionary Directory" button.

Note

To add another standard dictionary, such as ones found at the OpenOffice Dictionaries site, download, extract, and copy the files, e.g. en_GB.aff, en_GB.dic, hyph_en_GB.dic to the standard directory location and restart Sigil.

User Dictionaries

You can also create your own personal dictionaries, and even use different ones for different books.

To create a personal dictionary, select Add under User Dictionary on the left and then enter a name for your new word list:

You can choose which User Dictionary is in use, and you will see the words for that dictionary on the right. You can also rename or remove the dictionary.

On the right side you can add/edit/remove words in the selected Dictionary.

Tip

Whenever you use Add to Dictionary in Code View, your selected word will be added to the currently selected User Dictionary.
