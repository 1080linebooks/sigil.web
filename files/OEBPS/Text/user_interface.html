User Interface

— Windows, Menus and Toolbars —

Sigil provides several ways to use its features and run commands: main menu entries (like File→Open), icons on toolbars, keyboard shortcuts (like Ctrl+S to save your file), and Context Menus (shown when you right click on something like a filename).

The main actions are shown below, but you should refer to individual chapters in the guide for more details, including Book View icons and context menu options.

Note

Since Sigil runs on many OS's the screenshots in the User Guide may not exactly match what you see on your computer.

Windows

Sigil uses various windows in the main interface to present information: a window to show your files, to edit your book, to view your table of contents, etc.

Certain windows can be moved. For example, you can move the Table of Contents window to the left under the Book Browser instead of on the right, and you can even undock some windows (separate them from the main Sigil window):

To move, dock or undock a window click on the heading of the window and drag it to its new location. (In some OS's you can also undock and re-dock the windows by clicking on the little icon in the top right corner of the Book Browser or TOC window.)

Tip

To re-dock a separate window, double click on its title bar.

To hide or show specific windows use the View menu.

Main Menu

This is the list of menu items that is always at the top of Sigil's window or your screen:

File

  • New:
    • New Book: create a new, empty EPUB
    • Blank Section: create a new HTML file in the book
    • Blank Stylesheet: create a new CSS Stylesheet in the book
    • Add Existing File(s): add HTML files, images, etc. from your computer
  • Open: open an EPUB or HTML file from your computer – see the Open and Save Files chapter
  • Save: save your file (use often!)
  • Save As: save your file under a different name
  • Print Preview: preview the printing of the current tab
  • Print: print the current tab
  • Validate EPUB: check if your book meets EPUB standards (see Validate and Clean)
  • List of recently opened files
  • Quit: exit Sigil, asking if you want to save your file first if it hasn't been saved

Edit

  • Undo: restore text just deleted, inserted or replaced
  • Redo: reverse the last Undo
  • Cut: remove the selected text and store it for paste
  • Copy: copy the selected text and store it for paste
  • Paste: insert previously cut or copied text into your document
  • Metadata Editor: open the Metadata Editor – see the Metadata Editor chapter
  • Insert Image: insert an image in the EPUB into the text – see the Images chapter
  • Split Chapter: split the current file into 2 files at the end of the paragraph – see the Splitting and Merging chapter
  • Insert Chapter Marker: insert a chapter marker to use for splitting
  • Split at Chapter Markers: split the current file at inserted chapter markers
  • Clean Source: see the Validate and Clean chapter (you must restart Sigil for new settings to take effect)
    • Off: no automatic cleaning of code will be done
    • Pretty Print: clean up spacing and layout of code but do not change any code
    • Tidy: clean up code, convert some formats to inline styles, etc.
  • Check For Well-Formed Errors: check for invalid HTML – see the Validate and Clean chapter
  • Preferences: open the Preferences window – see the Preferences chapter

Search

  • Find / Replace: open the Find & Replace window – see the Find & Replace chapter
  • Find Next: search forwards for the next matching text
  • Find Previous: search backwards for the next matching text
  • Replace Next: replace any matched text and search forwards for the next matching text
  • Replace Previous: replace any matched text and search backwards for the next matching text
  • Replace All: replace every matching text according to the Find & Replace settings
  • Count All: count every matching text according to the Find & Replace settings
  • Go to Line: jump to a specific line number in Code View

View

  • Toolbars
    • File: show or hide the open and save icons
    • Edit: show or hide the undo and cut icons
    • View: show or hide the Book and Code view icons
    • Insert: show or hide the insert chapter and image icons
    • Donate: show or hide the Donate icon
    • Quick Settings: show or hide the quick settings icons
  • Book View: switch to Book View for editing
  • Preview View: switch to Preview View for reviewing
  • Code View: switch to Code View for editing
  • Zoom In: increase the size of text and images in the view
  • Zoom Out: decrease the size of text and images in the view
  • Zoom Reset: reset the window zoom to 100%
  • Book Browser: open or close the book browser window
  • Validation Results: open or close the validation window
  • Table of Contents: open or close the TOC window

Window

  • Next Tab: change view to the following tab
  • Previous Tab: change view to the preceding tab
  • Close Tab: close the current tab
  • Close Other Tabs: close all tabs other than the current tab
  • Open Previous File: open the file above the selected file in the Book Browser
  • Open Next File: open the file below the selected file in the Book Browser

Help

  • User Guide: the official Sigil User Guide
  • Frequently Asked Questions: common questions about Sigil
  • Tutorials: step by step guides to using Sigil
  • Donate: how to donate to ongoing development
  • Report an Issue: how to submit a bug report
  • Sigil DevBlog: occasional updates about development
  • About: Sigil version and author information

Main Toolbar

This is the list of icons that are always at the top of Sigil's window:

Tip

You can move sets of icons around to change their order or to place them to the left or right of the main window. Click and hold on the small bar to the left of each group of icons, then drag it to where you want it.

Right click on the toolbar to easily select which icons or windows to show.

The icons are listed below with their equivalent menu entry.

  • New: File→New→New Book
  • Open: File→Open
  • Save: File→Save
  • Save As: File→Save As

  • Undo: Edit→Undo
  • Redo: Edit→Redo
  • Cut: Edit→Cut
  • Copy: Edit→Copy
  • Paste: Edit→Paste
  • Find & Replace: Search→Find / Replace

  • Book View: View→Book View
  • Preview View: View→Preview View
  • Code View: View→Code View

  • Split Chapter: Edit→Split Chapter
  • Insert Image: Edit→Insert Image

  • Donate: Help→Donate

  • Check For Well-Formed Errors: Edit→Check For Well-formed errors
  • Validate: File→Validate Epub

Keyboard Shortcuts

Keyboard shortcuts (e.g. Ctrl+S for Save) are shown next to menu entries and in Keyboard Shortcuts in Edit→Preferences.

You can change the default keyboard shortcuts in Preferences.

Context Menus

Context Menus are menus that appear when you right-click on something – like a file in the Book Browser or a misspelled word in Code View. Context menu items are discussed in related chapters.

Tip

When in doubt, right-click on something to see if there are additional actions available!

Zoom

On the lower right of the main window is a zoom control that lets you shrink or enlarge the text and images in the main view:

The 100% indicates the zoom level of the view and this percentage will increase or decrease when you click on the + or – icons.

Status Bar

The bottom area on the main window (to the left of the zoom control) will occasionally list status information when an action is completed, like File Saved.

The status bar can also show information about the currently opened file, such as Line and Column in Code View.

Tooltips

Many of the buttons, dropdown boxes, labels and fields have tooltips that provide information on how they are used.

To see a tooltip just hover your mouse over the button or other item and the help text should be displayed.
