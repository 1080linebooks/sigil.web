Overview

— Sigil Highlights —

This is a quick summary of features in Sigil.

Editing

Sigil provides two ways to edit your book – Book View and Code View. The Book View editor is a WYSIWYG editor (based on CKEditor [http://ckeditor.com]) which is useful when you need to make basic changes. The Code View editor on the other hand allows you to directly edit the HTML formatting of your files when you want complete control over your book or you want to use Sigil's advanced features. There is also a read-only Preview View which allows you to test modifying styles and to inspect your code in detail. Tabs are opened for each of the files in the EPUB that you edit, and you can even magnify the text. Spell checking with multiple dictionaries is also available.

Find & Replace

The ability to search for or replace text is one of the most powerful functions in Sigil. Book View provides a basic find function, but Find & Replace is usually used in Code View where all its features are available and it can be used to change or correct formatting and styles. You can search or replace in one file or in all the files at once, and you can search for simple text or use Regular Expressions.

Managing Files

The Book Browser organizes all your files by type and gives you quick access to open files or to modify them. Double-click a file to open it in the editing window, or right-click on selected files to see options for renaming, deleting, merging, adding, linking stylesheets, etc. The HTML files are listed in the order they appear in your book – and you can drag and drop them to reorder them.

Table of Contents

The Table of Contents (TOC) window shows you exactly what your TOC will look like, and allows you to jump to specific chapters by clicking on chapter names. You can also generate a new TOC based on headings in your book at any time, and create an HTML version of your TOC.

Modifying Book Details

The Metadata Editor provides a quick way to update book details like Title, Author, and Language. You can add any of the hundreds of standard EPUB metadata entries like publication date, co-authors, copyright information, etc.

Preferences

Many settings in Sigil can be customised by opening the Preferences window. In particular you can set Sigil's user interface language, keyboard shortcuts and dictionaries.

Validation

Sigil performs automatic checking of your code to highlight errors when saving, and it includes the FlightCrew EPUB validator [http://code.google.com/p/flightcrew/] to help ensure your book conforms to EPUB standards. Sigil can also automatically clean and format your code using HTML Tidy [http://tidy.sourceforge.net/].

Language Support

Sigil has been translated into over 15 languages and makes it easy to change the user interface language as needed. It is written in C++ for high performance and includes full Unicode UTF-16 support.
