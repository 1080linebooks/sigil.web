Convert Your File For Import

If you already have an EPUB or HTML file you can skip this step.

If you are starting with a document created by one of the many available word processor programs, then the first thing you need to do is convert your document to HTML so that you can import it into Sigil.

The "cleaner" your HTML file is, the easier it will be to turn it into an EPUB and the more likely it will display correctly on multiple devices. This means not using too many fancy formats in your document, and preferably using styles for formatting.

Although conversion is not part of Sigil, here are some suggestions for converting files for import into Sigil.

Convert Word to HTML

Use the MS Word menu option File→Save As Filtered HTML. Or use the MS Word Macro @ MobileRead.

Both approaches try to reduce the amount of extra code exported by MS Word so that your HTML file is cleaner and simpler to edit in Sigil. You really want to have your code as clean as possible since Word adds lots of extra HTML tags that you will probably want to delete at some point.

When using Word or any word processor, you should always try to use styles to format your code instead of just making some text bold, some larger, etc. Styles are also an important concept when formatting EPUBs.

Convert Text to HTML

It's not strictly necessary to convert text files to HTML. You can just paste the text into Sigil's Book View. In fact, Sigil can also directly open .txt files.

But it's worth noting that the problem with text files is that they don't always convert well because the paragraph separations are not clear. This lack of formatting means you can end up with lots of run-on paragraphs, etc. leaving you to do a lot of cleanup work. Text files are still a better source than some formats since they at least have fewer unnecessary HTML tags.

Convert Mobi to HTML

The Mobi/KF8 format is actually an HTML file inside of a zip file. So 'converting' it is actually quite easy – as long as the files are not protected with DRM.

Simply unzip the .prc, .azw, .azw3 file into a directory. You can now access the HTML file.

Convert Other Formats

You can also convert other formats to HTML or EPUB using other software such as Calibre.

Note that any automatic conversion of your files will introduce lots of extra code. If you are just formatting a book for personal use you may not be concerned with this. But if you are formatting books for professional purposes or want as little extra code in your EPUB as possible, then it should be avoided.

Sigil only supports importing HTML files since converting from other formats is actually quite specialized and best left to other tools.
