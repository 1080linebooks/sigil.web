About EPUB

The EPUB (electronic publication) format is one of the most popular formats for ebooks. It is an open and freely available format that can be used by anyone.

An EPUB ebook is just a file containing several other files stored together using the common zip format. These files include your words, images, Table of Contents, stylesheets, fonts and details about your book like author or title. EPUB's standard format means your book can be read on many ereaders.

Words and text are actually stored in the EPUB book using the HTML format (Hyper Text Markup Language – the same code used by web pages to display text and images). HTML files contain your text along with markup codes and tags (like <h1> or <p>) which tell ereaders how to format and display the text – some words could be bold or italic, others could be a paragraph or a heading, etc.

Although you can use only HTML tags to control or style the layout, EPUB books typically make use of stylesheets to help organize the use of styles used by the HTML file. A stylesheet is another file in the EPUB that contains a list of HTML tags along with instructions on how to display any text using that tag, e.g. <h1> (heading level 1) should be larger and centered. This allows you to separate the content of your book from its layout and allows you to easily adjust the look of the EPUB.

Of course, when anyone actually reads an EPUB they don’t see all these HTML files or stylesheets. The ereader just displays the text and images like a normal book as close as it can to the formatting you defined, even if there are thousands of pages or files.

You don't have to understand all the details of the EPUB format or understand HTML formatting codes to start using Sigil. However, if you spend any time with Sigil you will soon need to become familiar with these areas in order to get the most out of the advanced features of Sigil, and, ultimately, to achieve the design you want.

Note

Technically the text files are in XHTML format (an XML compliant version of HTML), but to keep things simpler we'll use HTML to refer to either HTML or XHTML in this guide.

To learn more about HTML you can read various online guides, including:

  • W3Schools HTML Guide

Help with EPUB is available from EPUB users at MobileRead:

  • EPUB Forum @ MobileRead

If you really want to dig into the details of the official EPUB standard on which Sigil is built you can read the original technical specifications:

  • Official EPUB 2.0.1 IDPF Standard
    • OPS (Open Publication Structure): the key specification which describes everything in an EPUB file
    • OPF (Open Packaging Format): how the EPUB files relate to each other, metadata details (content.opf)
    • OCF (OEBPS Container Format): how files are packaged into a .epub

A great way to learn about EPUB is to find an EPUB ebook that is formatted how you like, load it into Sigil and look at it.
